## Project Overview

The Infrastructure Automation Repository manages:

1. **Cloud Infrastructure Provisioning**: Creating and managing cloud resources using Terraform.
2. **Configuration Management**: Configuring cloud resources and deploying necessary software with Ansible.
3. **GitOps Workflows**: Enabling declarative infrastructure management for seamless integration with CI/CD pipelines.
